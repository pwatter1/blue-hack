# IBM Blue Hack 2017 

BlockPay - Blockchain for group financing.

## Technologies
BlockPay is built on a Node.JS server, interacting with IBM Bluemix Blockchain and a Python backend.

## Features 
### App
- Create a charge
- Submit single payment
- Submit group payout
### Blockchain
- List new item
- Approve item
