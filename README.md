# IBM Blue Hack

BlockPay - Blockchain for group financing. <br />
Tied for 1st in the Cognitive solutions and Blockchain category at IBM Research in Yorktown Heights, NY. <br />
https://www.blueopen.org/home

## Technologies
- Node.JS server
- Hypervisor Blockchain 
- Python backend

## Features 
#### App
- Create a charge
- Submit single payment
- Submit group payout
- Login/Logout
- User profile/accounts

#### Blockchain
- List new item
- Approve item
